Write the difference between var, let and const with code examples.

var,let,const

1. "var" declaration are globally scoped function while "let" and "const" are block scoped"
{
var name = "Ghrushma";
console.log(name); //this will print name in the console
} console.log(name); //this will print name in the console
{

console.log(value); //this will print value in the console
}
//console.log(value); //this will give error.
{
